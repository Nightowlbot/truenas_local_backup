network.general.summary
=======================

Retrieve general information for current Network.

Returns a dictionary. For example:

.. examples(websocket)::

    :::javascript {
        "ips": { "vtnet0": { "IPV4": [ "192.168.0.15/24" ]
            } }, "default_routes": [ "192.168.0.1" ], "nameservers": [ "192.168.0.1" ]
    }
