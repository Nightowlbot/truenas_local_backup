certificate.update
==================

Update certificate of `id`

Only name attribute can be updated.

.. examples(websocket)::

  Update a certificate of `id`

    :::javascript {
        "id": "6841f242-840a-11e6-a437-00e04d680384", "msg": "method", "method": "certificate.update", "params": [ 1, {
                "name": "updated_name" }
        ] }

This method is a job.
